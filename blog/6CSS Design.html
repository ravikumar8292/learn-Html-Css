CSS Design

CSS Design

  1. CSS Grid

    2. A grid can be defined as an intersecting set of horizontal lines and vertical lines. CSS Grid layout divides a page into major regions. It defines the relationship between the parts of a control built from HTML primitives in terms of layer, position, and size. Grid property offers a grid-based layout system having rows and columns. It makes the designing of web pages easy without positioning and floating.

    Grid Container

    We can define the grid container by setting the display property to grid or inline-grid on an element.

    Let's see some of the shorthand properties:

    1. grid-template-columns: It is used to specify the size of the columns.
    2. grid-template-rows: It is used to specify the row size.
    3. grid-template-areas: It is used to specify the grid layout by using the named items.
    4. grid-auto-rows: It is used to specify the automatic size of the rows.
    5. grid-auto-columns: It is used to specify the automatic size of the columns.
    6. grid-auto-flow: It is used to specify how to place auto-placed items and the automatic row size.

  2. CSS Layout

    3. CSS layout is easy to design. We can use CSS layout to design our web page such as home page, contact us, about us etc.

    There are 3 ways to design layout of a web page:

    1. HTML Div with CSS: fast and widely used now.
    2. HTML Table: slow and less preferred.
    3. HTML Frameset: deprecated now.

  3. CSS Table

    4. We can apply style on HTML tables for better look and feel. There are some CSS properties that are widely used in designing table using CSS:

    1. border
    2. border-collapse
    3. padding
    4. width
    5. height
    6. text-align
    7. color
    8. background-color
